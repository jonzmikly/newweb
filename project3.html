Project 3 - Hobby or Organization Site

Objective: Use Dreamweaver and Photoshop to make a site for a hobby or organization.

Requirements:

  • You will use Twitter Bootstrap and Photoshop in designing this project.
  • Use the Responsive Design files, provided by Bootstrap, and create your own index page. Attach the bootstrap.css stylesheet and create a general structure for this site.
  • Use divs for formatting each page. Each page must have a header, navigation and content within a columned grid.
  • Site must have a total of at least four pages. Pages can represent separate areas of the site, like home, about, contact, links, etc.
  • Decide on an appropriate navigation strategy. Drop-downs are optional.
  • Include at least one or more appropriate photos or graphics on each page.
  • Design a site logo, using Photoshop. If the organization has a logo, you may use it, but you will also be required to make another graphic somewhere else in the site. You can put this logo either in the header or in your navigation list (as explained in Bootstrap Handout)
  • Add at least two "Components" to your site. This could be buttons, glyphicons, labels, alerts, etc. Navigtation does not count as a component unless you add a dropdown.
  • Include at least three external links to relevant sites.
  • Make sure you include your email address somewhere in site.
  • Add a date to the bottom of the home page. Update this every time you make a change.
  • Make sure you save all pages with a .html extension.
  • Upload all associated files to a new directory of your website (ex: jonzmikly.com/fishing)
  • Make sure this project is linked to your Assignments page (external link).

For examples, check out the Student Work page.
